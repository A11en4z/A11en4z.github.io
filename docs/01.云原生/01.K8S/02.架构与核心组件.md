# 架构与核心组件

## 控制面（Control Plane）

| 组件 | 作用 |
| --- | --- |
| kube-apiserver | 集群 API 入口，所有操作经此 |
| etcd | 持久化集群状态 |
| kube-scheduler | 为 Pod 选择 Node |
| kube-controller-manager | 运行各类控制器，维持期望状态 |

## 节点（Node）

| 组件 | 作用 |
| --- | --- |
| kubelet | 与控制面通信，管理本机 Pod |
| kube-proxy | 维护 Service 网络规则 |
| 容器运行时 | containerd / CRI-O 等 |

## 常用资源对象

- Namespace
- Pod / ReplicaSet / Deployment
- Service / Endpoints
- ConfigMap / Secret
- Ingress
- PersistentVolume / PersistentVolumeClaim
