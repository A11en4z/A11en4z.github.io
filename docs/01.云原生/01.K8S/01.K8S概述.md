# K8S 概述

> 将你的 Obsidian「K8S」文件夹内容替换本页及同目录下其他笔记即可。

## 什么是 Kubernetes

Kubernetes（常简称 K8S）是一个开源的容器编排平台，用于自动化部署、扩缩容和管理容器化应用。

## 核心能力

- **调度**：把 Pod 放到合适的 Node 上
- **自愈**：容器崩溃时自动重启 / 重建
- **扩缩容**：按负载调整副本数
- **服务发现与负载均衡**：Service / Ingress
- **配置与密钥**：ConfigMap / Secret
- **存储编排**：PV / PVC / StorageClass

## 建议学习路径

1. 容器基础（Docker）
2. 集群架构（Master / Node / 控制面组件）
3. 工作负载（Pod / Deployment / StatefulSet / DaemonSet）
4. 网络（CNI / Service / Ingress）
5. 存储与配置
6. 调度、权限（RBAC）、可观测性
